Reglas Fundamentales

Escala de espaciado
Solo múltiplos de 4px. Nada arbitrario.
Border radius mínimo
Mínimo 8px. Nunca elementos cuadrados perfectos.
Pesos tipográficos
Máximo 3 por vista: Regular, Semibold, Bold.
Sombras sutiles
Opacidad máxima 0.16. Siempre difuminadas.
Estados visuales
Todo elemento interactivo tiene hover/active/focus.
Touch targets
Mínimo 44x44px para accesibilidad móvil.
