# PROMPTS GENERALES - ANÁLISIS Y RECOMENDACIONES (MÁXIMO 811 CARACTERES)

## PROMPT #1: Análisis General de Seguridad y Arquitectura (625 caracteres)

```
Analizar la integración completa de Supabase en el frontend React de ComerECO WebApp. Evaluar seguridad (claves hardcodeadas, sanitización de queries, filtros real-time), autenticación (patrones de uso, refresh tokens), manejo de errores (consistencia, mensajes), real-time (cleanup, memory leaks), validaciones (permisos cliente, datos), configuración (variables entorno), y arquitectura (contextos, servicios). Identificar problemas críticos, importantes y mejoras. Proporcionar recomendaciones prioritarias con soluciones concretas para cada problema encontrado. Enfocarse solo en código frontend, no modificar backend/BD.
```

## PROMPT #2: Análisis de Problemas Críticos y Mejoras Prioritarias (297 caracteres)

```
Revisar la integración de Supabase en el frontend React. Identificar los problemas más críticos en seguridad, autenticación, manejo de errores, real-time y validaciones. Priorizar por impacto en seguridad y funcionalidad. Proporcionar recomendaciones concretas y ejecutables para las mejoras más urgentes. Solo código frontend.
```

---

## USO

Estos 2 prompts están diseñados para generar análisis generales y recomendaciones:

- **Prompt #1**: Análisis completo y exhaustivo de toda la integración
- **Prompt #2**: Enfoque en problemas críticos y mejoras prioritarias

Ambos prompts están bajo 811 caracteres y pueden usarse directamente con Horizon AI o cualquier asistente de IA.
